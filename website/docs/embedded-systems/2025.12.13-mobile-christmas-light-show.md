## Problem

In 2024, my NSBE chapter participated in the Montgomery Christmas Parade. This was the first 
year of doing it and we were not prepared. 

## Solution

In 2025, I decided to do a mobile or portable version of the Christmas light show that I do at my house. 
It was very much scaled down, because you cannot put that much stuff on a vehicle. 
I was able to use the same songs, but added additional elements and props to the existing sequences
so that I did not have to reduce the entire show setup.

## Controller Software 

Falcon Pi Player is used to control the lights. Sequences are loaded to the Raspberry PI and are started
using the web interface of FPP.

## Song List

* All I Want for Christmas - Lil Jon f/ Kool-Aid Man
* Baby Shark
* Nutrocker - Trans-Siberan Orchestra
* Christmas Time Is Party Time - Luke


## Other Details

Prior to participating in the parade, we use the radio station locator, to find a vacant frequency to 
transmit the audio over. Due to the signal not being strong enough, the sound system would not play the 
audio via FM radio, because the signal strength was strong enough.  Thus had to rely on the vehicle 
sound system to play the music.

For future parades, I have considered using a 12V set up, so that the setup can be connected to the 
vehicle's electrical system and not require a generator. The major change would be the lights have to 
be replaced with 12v lights.  The inflatable uses 12V output from the provided power brick, so it could 
connect directly to the power source.
